# This Project is a Test


## the project establishes a solid ground for Network Incident Tracking 


To run this project:
 - Git clone this repo
 - establish a python virtual env using `python -m venv venv`
 - install the requirements.txt
 - run database.py to create the db
 - run seed.py to seed the db
 - run this command to run the app `flask --app app.py run`
